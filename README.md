## API Endpoints

| HTTP Verbs | Endpoints        | Action                                     |
|------------|------------------|--------------------------------------------|
| GET        | /api/sensor/:id  | Get sensor information for given Sensor id |
| GET        | /api/sensor/     | Get all sensors                            |
| GET        | /api/sensorsData | query sensor data                          |
